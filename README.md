# Adaptive Health Systems Planning under Uncertainty

## Overview
This repository provides a prototype simulation model supporting the study:

"Adaptive Health Systems Planning under Uncertainty: 
An Agent-Based and System Dynamics Integrated Framework 
for Policy Intelligence and Resilience"

The model integrates:
- Agent-Based Modeling (ABM)
- System Dynamics (SD)
- Network interactions
- Adaptive Policy Intelligence Layer

## Purpose
This code is intended for:
- Conceptual demonstration
- Transparency and reproducibility
- Appendix support (not empirically calibrated)

## Key Features
- Multi-level system modeling (micro, meso, macro)
- Dynamic feedback mechanisms
- Policy adaptation through learning
- Scenario simulation (pandemic, resource constraints, digital transformation)

## Requirements
Python 3.8+
